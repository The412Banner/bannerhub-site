BannerHub

The full-featured flagship. BannerHub brings deep GOG / Epic / Amazon integrations (DLC, cloud saves, EOS online auth), an in-app downloads manager, AI Frame Generation, PC-accurate rumble, front-end export and a thriving community-config library — all on the mature GameHub 5.3.5 ReVanced base.

What's in BannerHub

🎮 Store integrations

  • GOG — OAuth2, Gen 1 & Gen 2 downloads, SHA-256 chunk verification, multi-CDN failover, cloud saves, per-game DLC.
  • Epic — OAuth2, catalog sync, 6-parallel chunked downloads, free-games section, full-screen detail pages, Epic Online Services online auth, DLC.
  • Amazon — PKCE OAuth2, GetEntitlements sync, manifest.proto parsing, FuelPump launch, SDK DLL deployment, DLC.
  • Update checker, install-size & release-date display, and gamepad navigation across all store tabs.

⚙️ Component Manager

  • In-app downloader for DXVK, VKD3D, Box64, FEXCore and GPU drivers (Turnip).
  • Multiple repository sources including The412Banner Nightlies.
  • Full .wcp / .zip component control — add, inject, back up, remove.
  • One-click install of any components a community config requires.

🚀 Performance & control

  • AI Frame Generation — in-game menu, 6 quality presets, 2×/3×/4× multiplier, custom flow scale (≈1.8–1.9× FPS on supported Adreno).
  • PC-accurate Vibration / Rumble — independent dual-motor XInput rumble, sustained holds, instant release, multi-controller.
  • Winlator HUD — Normal / Extra-Detailed / Konkr display modes. RTS touch controls, CPU core affinity, VRAM unlock (6/8/12/16 GB), sustained-perf & GPU-clock toggles.

🗂️ Downloads, configs & tooling

  • Background download service + in-app cross-store download manager with live progress & notifications.
  • Community Config System — search / view / download / export per-game configs; 7,000+ device-specific configs across 1,700+ games; no account needed; cross-compatible with Lite. Browse them at the community config site (repo).
  • Wine Task Manager — Container Info, Applications & Processes (kill), and a prefix Launch browser.
  • Front-end export (Beacon / ES-DE), offline launch for imported games, Japanese localization, dynamic versioning.

📦 Variants & packaging

Nine APK variants install side-by-side via different package names — Normal (banner.hub), Normal-GHL (gamehub.lite), PuBG, AnTuTu, alt-AnTuTu, PuBG-CrossFire, Ludashi, Genshin and Original. Performance-spoof variants push the device harder (more heat) — use with proper cooling. All signed with a consistent key for in-place updates.

🧬 Inherited from the base

Built on the GameHub 5.3.5 ReVanced base

BannerHub is built on the GameHub 5.3.5 ReVanced base maintained by PlayDay (playday3008/gamehub-patches, GPLv3) — a set of ReVanced Patcher modifications to XiaoJi GameHub. Much of BannerHub's "core" behaviour comes from that patch lineage; BannerHub layers its store integrations, AI Frame Generation wiring, vibration and config system on top.

From the ReVanced patch set

  • Bypass login — use the app with no account.
  • Privacy patches — disable analytics, push, heartbeat playtime, OTA updates & cloud timer; manifest cleanup & promo removal.
  • Mute UI sounds, file-manager access provider, and an external launcher hook.
  • Steam patches — force Steam TCP, force Steam input, Steam CDN, SD-card storage, Steam playtime.
  • Performance-metrics overlay and RTS touch controls.
  • Game-ID display, hide-recommended-games, compatibility fallback, account-currency, settings menu, package/app rename.

…and from GameHub 5.3.5 itself

  • The Wine + Box64 / FEX runtime for Windows games.
  • Container / Wine-prefix management and the in-game sidebar.
  • Native Steam client integration (the 5.3.5 base is pinned for Steam-card support).
  • The DXVK / VKD3D / Box64 component pipeline.

BannerHub then adds: GOG / Epic (incl. EOS) / Amazon stores with DLC & cloud saves, the in-app download manager, AI Frame Generation, PC-accurate rumble, front-end export and the community-config library (see above).

Credit & lineage: GameHub Patches © PlayDay (playday3008), GPLv3; ReVanced framework © ReVanced; GameHub © XiaoJi. The upstream patch project evolves independently — this lists its GameHub-targeted feature patches as the inherited base.
