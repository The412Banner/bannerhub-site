Why use BannerHub instead of stock GameHub?

Same engine you already trust to run Wine — minus the gatekeeping and the tracking, plus the tools power users actually want.

🔓

No login wall

Launch straight to the home screen and import / play your own games — no XiaoJi account, no phone-number login, no region gate.

🛡️

Privacy-hardened

v6 strips Firebase, Mob Push, GMS Measurement, Ad-ID, OTA & analytics at the bytecode level — and honestly documents what's left.

🎮

More stores

First-class GOG, Epic and Amazon libraries with downloads, DLC and cloud saves — alongside the existing Steam path.

⚙️

Curated components

Hand-picked DXVK, VKD3D-Proton, Box64, FEX and Mesa Turnip builds delivered through the BannerHub component channel.

🚀

Power-user tooling

AI Frame Generation, PC-accurate controller rumble, Winlator HUD, Wine Task Manager, GPU spoofing and an in-game performance overlay.

🧩

Front-end ready

Launch your games from Beacon, ES-DE, Daijishou, NeoStation and more — BannerHub ships a documented intent contract for every variant.

Straight with you: BannerHub patches the GameHub app — it doesn't change the Windows games you run, the Wine / Box64 stack, or Valve / Epic / GOG's own services. Compatibility differs between the three versions, and v6 is still a work in progress. Everything here is open source so you can verify the claims yourself.
